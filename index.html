Superstore Business Performance Analysis

Comprehensive analysis of sales data to drive business strategy

Project Demonstration

This video walkthrough demonstrates the interactive features of the Superstore dashboard and key findings from the analysis.

Dashboard Views

Project Description

The Superstore Business Performance Analysis project aims to uncover actionable insights into sales performance, profitability, and customer behavior across regions and product segments. The workflow began with data cleaning and preprocessing using SQL and Python (Pandas), followed by visual exploration with Plotly Express. Tableau was used to design an interactive dashboard based on a custom layout sketched in Draw.io and styled using Figma for visual consistency. The project was documented and planned in Notion for organized development. The final dashboard allows business stakeholders to monitor KPIs, identify underperforming areas, track seasonal trends, and make data-driven decisions that support growth and efficiency.

Project Steps

  • Data Collection: Gathered 4 years of sales data from Superstore's database
  • Data Cleaning: Addressed missing values, outliers, and inconsistencies using Python
  • Exploratory Analysis: Identified initial patterns and correlations in the data
  • Dashboard Design: Created wireframes in Figma for optimal user experience
  • Visualization Development: Built interactive dashboards in Tableau
  • Insight Generation: Derived actionable business recommendations

Key Insights

  • Category Performance: Technology category showed 15% quarter-over-quarter growth, while Furniture had lowest profit margins at 8%
  • Regional Analysis: West region consistently outperformed others with 23% higher profit margins
  • Customer Segmentation: Corporate clients accounted for 45% of total sales but only 35% of profits
  • Seasonal Trends: Q4 sales were 40% higher than other quarters due to holiday season
  • Shipping Analysis: Standard shipping had 15% higher customer satisfaction than express options

Recommendations

  • Reallocate marketing budget to focus on high-growth Technology category in West region
  • Implement pricing strategy review for Furniture category to improve margins
  • Develop targeted promotions for Corporate clients to improve profitability
  • Expand inventory planning for Q4 peak season based on historical trends
  • Investigate reasons behind lower satisfaction with express shipping options

Road Accident Analysis

Interactive dashboards for transportation safety planning

Project Description

This project analyzes road accident data to uncover trends and patterns that contribute to transportation safety planning. Using Excel, Power Query, Power Pivot, and Pivot Tables, I developed an interactive dashboard that breaks down accident data by time, location, weather, road conditions, and driver characteristics. The goal was to provide a dynamic reporting tool for decision-makers to understand accident hotspots and identify high-risk factors. Key insights such as peak accident times, urban vs. rural accident rates, and the impact of weather were highlighted to support actionable safety recommendations.

Project Steps

  • Data Collection: Compiled accident reports from national transportation database
  • Data Transformation: Used Power Query to clean and reshape raw data
  • Data Modeling: Created relationships between tables using Power Pivot
  • KPI Definition: Established key metrics for accident severity and frequency
  • Dashboard Development: Built interactive views with slicers for user exploration
  • Validation: Cross-checked findings with official statistics for accuracy

Key Insights

  • Temporal Patterns: Friday evenings (6-9pm) showed 28% higher accident rates
  • Location Analysis: Urban areas accounted for 65% of accidents despite better infrastructure
  • Weather Impact: Weather-related accidents peaked during winter months (Nov-Feb)
  • Vehicle Types: Motorcycles involved in 40% of fatal accidents despite being 15% of vehicles
  • Road Conditions: Wet surfaces contributed to 35% more severe accidents than dry conditions

Recommendations

  • Increase police patrols during high-risk Friday evening hours
  • Implement targeted urban safety campaigns focusing on pedestrian crossings
  • Develop winter weather preparedness program for drivers
  • Create motorcycle safety initiative with mandatory training components
  • Improve drainage systems on roads with frequent wet surface accidents
